# Habeebullah's Professional App

A professional Streamlit application created at the University of Lagos.

## 🚀 Features

- **User-friendly interface** with personalized greetings
- **Service selection** - Choose from multiple services:
  - App Development
  - Data Analysis
  - Digital Design
  - Just saying Hi!
- **Interactive dashboard** with responsive design
- **Development mode** settings in the sidebar

## 📋 Requirements

- Python 3.7 or higher
- Streamlit 1.0 or higher

## 🛠️ Installation

1. Clone the repository:
```bash
git clone https://github.com/1Adee/habeebullah-streamlit-app.git
cd habeebullah-streamlit-app
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## ▶️ Running the App

Run the application with:
```bash
streamlit run app.py
```

The app will open in your browser at `http://localhost:8501`

## 📝 Usage

1. Enter your name in the text input field
2. Select a service from the dropdown menu
3. Click the "Launch App" button
4. See your personalized welcome message!
